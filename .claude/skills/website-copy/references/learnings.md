## How to Use This File

After launching or testing website copy:

1. **Document what converted** - Headlines, CTAs, or sections that performed
2. **Document what didn't work** - Copy that underperformed expectations
3. **Note A/B test results** - What won and why
4. **Capture user feedback** - What users said about clarity
5. **Record "Grandma Test" results** - Clarity insights
